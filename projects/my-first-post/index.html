Timetable Application

The application creates a personalised Timetable for College based on your Core subjects, Electives and Subdivision specific Labs. This Timetable can be resized and edited. It can then be downloaded.

The Timetable uses html2canvas to take a screenshot of the Timetable. It uses Javascript and JQuery to add the editing options.
